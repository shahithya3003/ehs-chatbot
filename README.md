# EHS Chatbot for Dynamic Data Querying

## Description
A full-stack AI chatbot that converts natural language queries into SQL queries
to retrieve real-time EHS insights.

## Tech Stack
- Backend: Django, PostgreSQL
- Frontend: React
- AI: Phi-3 Mini LLM
- Database: PostgreSQL

## Features
- Natural language to SQL conversion
- Real-time dashboard insights
- Secure role-based access
